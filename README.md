## What I'm doing now?

### 1) Travel, job applications, and devops project

I'm off to Europe and starting to put myself out there for new roles. Still doing all the things I love: learning, reading, exercising, cooking, exploring. But I'm turning up the dial on job applications now.

To help with this process I'm working on an end-to-end demo project that showcases my DevOps skills. This involves building a simple coffee concession card app that gives me a realistic, non-trivial application to push through a full DevOps pipeline from code to production. The app itself is deliberately simple: a few endpoints, a database, some light business logic. It's actually inspired by [Raglan Roast's](https://raglanroast.co.nz/) coffee loyalty book, check them out if you're ever in NZ!

The real work is everything around it — CI/CD, infrastructure as code, container orchestration, GitOps, environment promotion — split across three repos to mirror how teams actually structure things in the real world. 

The goal is a portfolio piece that shows I can take something from a local dev environment all the way to production on AWS (and eventually GCP), using modern practices, and explain every decision along the way.
